# CHANGELOG

## Unreleased

+ [新增] 新增用户账号与算力点体系，支持账号密码注册登录、Linux.do OAuth 登录、用户管理、算力点调整和算力点日志。
+ [新增] 后端模型渠道调用支持按模型算力点计费，生图、图生图、文本对话和视频接口会在请求前预扣算力点，上游失败时自动返还。
+ [新增] 管理后台公开配置支持设置模型算力点、默认图片模型、默认视频模型和默认文本模型。
+ [新增] 画布右上角展示当前用户算力点余额，节点发送按钮、右侧助手和生成配置节点会展示本次预计消耗算力点。
+ [新增] 新增视频生成节点，支持本地直连和云端渠道的视频生成、轮询、下载、保存到素材和参考图生成视频。
+ [优化] 画布模型选择器统一为共享下拉组件，云端模式使用后台公开配置的可用模型，并按模型名称展示常见厂商图标。
+ [优化] 画布图片和视频节点会按素材或生成结果的真实宽高比调整尺寸，空节点切换比例时也会同步更新外框。
+ [优化] 画布生图设置弹层改为紧凑面板，模型选择保留在底部工具栏，尺寸、比例、质量和生成数量配置更集中。
+ [优化] 管理后台渠道模型列表获取和模型测试改为走后端接口，不再由前端直接读取或转发 API Key。
+ [优化] 管理后台渠道可用模型选择器支持单行省略展示，临时移除模型后仍保留候选，方便重新添加。
+ [优化] 后台提示词定时同步默认开启并默认每 5 分钟执行一次。
+ [修复] 云端默认模型为空时前端会回落到第一个系统可用模型，避免模型下拉为空或误弹配置弹窗。
+ [修复] 模型算力点配置输入不再因失焦回跳为 0，系统可用模型临时删除时会保留历史算力点配置。

## v0.0.7 - 2026-05-23

+ [新增] 管理后台提示词管理支持多选批量删除。
+ [新增] 新增定义拉取GitHub提示词源功能。
+ [新增] 新增awesome-gpt-image2-prompts提示词来源。
+ [优化] 优化模型下拉选择样式、优化生图编辑设置

## v0.0.6 - 2026-05-22

+ [新增] 管理后台支持配置模型渠道，前端当前无需鉴权即可直接使用后端渠道能力。
+ [优化] 统一整理后端错误提示、AI 代理、图片节点生成与重试、参考图缺失处理等细节。
+ [优化] 后端模型代理路径调整为 OpenAI 风格。

## v0.0.5 - 2026-05-20

+ [新增] 右上角版本号支持点击查看版本更新弹窗，展示当前版本、最新版本和按时间线整理的更新日志。
+ [新增] 设置弹窗支持配置系统提示词，AI 生图、编辑图和文本请求会自动携带。

## v0.0.4 - 2026-05-20

+ [调整] Docker 运行入口改为 Next.js 对外提供页面，`/api/*` 由 Next.js 代理到内部 Go 服务。
+ [修复] 文本复制在局域网 IP 访问时可能失败的问题。

## v0.0.3 - 2026-05-19

+ [修复] 更新 nanoid 依赖并修改 ID 生成方式，防止其他ip无法使用crypto模块导致的ID生成失败问题。

## v0.0.2 - 2026-05-19

+ [新增] 增加生图工作台功能，支持文生图、图生图、查看历史记录，并增加移动端适配。
+ [修复] 画布生成尺寸控件支持选择更多常用比例，并可直接输入自定义比例。
+ [修复] 生成配置节点恢复拖拽操作，避免面板控件拦截整块节点拖动。
+ [文档] 增加 Render 部署说明。

## v0.0.1 - 2026-05-19

+ [新增] 首次开源版本，包含无限画布能力：多画布项目、节点拖拽缩放、连线、小地图、撤销重做、导入导出。
+ [新增] AI 创作能力：支持 OpenAI 兼容接口的文生图、图生图、参考图编辑和文本问答。
+ [新增] 画布助手能力：支持围绕选中节点和上游节点对话、生图，并把结果插回画布。
+ [新增] 提示词库能力：抓取多个 GitHub 开源项目，按案例整理数百个图片提示词。
